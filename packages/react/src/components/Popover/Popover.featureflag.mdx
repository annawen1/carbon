## Enable dynamic setting of floating styles

`Popover`, `Dropdown`, `Tooltip` supports the
`enable-v12-dynamically-set-floating-styles` feature flag. This enables a
third-party library, `floating-ui` to set the styles for the floating element.

```js
<FeatureFlags
  flags={{
    'enable-v12-dynamically-set-floating-styles': true,
  }}>
  <Popover open={open}>
    <button
      type="button"
      onClick={() => {
        setOpen(!open);
      }}>
      Toggle
    </button>
    <PopoverContent>
      The content that is revealed by interacting with the Toggle button
    </PopoverContent>
  </Popover>
</FeatureFlags>
```
